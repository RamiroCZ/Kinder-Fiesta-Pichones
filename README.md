# Kinder Fiesta

Sitio web para buscar, calificar y comentar sobre salones infantiles.  
---

## 1. Requisitos del Sistema

# Software Necesario
- Python **3.8 o superior**
- Pip (gestor de paquetes de Python)
- SQLite3 (incluido por defecto en Python)
- Navegador web moderno (Chrome, Edge, Firefox, etc.)

La estructura del proyecto incluye:

Carpeta static/ = CSS, imágenes y JS
Carpeta templates/ = archivos HTML
Archivo principal app.py
Archivo init_db.py para crear la base de datos SQLite3


## 2. Instalación de las Herramientas de Trabajo
pip install -r requirements.txt

# 1. Descargar o clonar el proyecto
Si usas Git:

git clone https://github.com/usuario/kinderfiesta.git
cd kinderfiesta

## 3. Crear y activar entorno virtual (recomendado)
Windows
python -m venv venv
venv\Scripts\activate

Linux/Mac
python3 -m venv venv
source venv/bin/activate

## 4. Inicializar la base de datos
python init_db.py

## 5.Ejecutar el servidor local
python app.py

## 6. Abrir el sitio web
http://localhost:5000
